# 运行设置

```
# 进入 music-pinia 目录
cd music-pinia
# 安装依赖包
npm install
# 运行项目
npm run dev

# 进入 music-server/nodejs 目录
cd ../music-server/nodejs
# 运行后台服务
node nodeserver.js
```

# 说明

此项目只有两首歌，在 music-server/music 目录下

并且音乐信息也只有两个json文件

所以只支持两首音乐的播放



登录和注册只是使用 node.js 存储在本地，并没有使用数据库功能



此项目使用node.js+vue3+pinia
